DDB Redirecter

Welcome to the DDB Redirecter! I hope you find what you're looking for.

Message from the other side...

What is this?

A self-contained redirecter to D&D Beyond. Enter a source and a page number and you'll redirected to the (closest) corresponding section on DDB.

What does it mean it's self-contained? All the code is within the page itself, with the exception of the stylesheet.

Why is it?

I'm a D&D newbie and started "all digital" with D&D Beyond. I found myself wanting to look up these page references that kept coming up here and there, but there wasn't an easy way of doing this. This isn't a huge problem, but I was interested to see if I could make a page reference mapped to DDB.

I started out by making a DDB to physical page reference, meaning I went from the table of contents on DDB and checked which physical pages they represented.

It wasn't an optimal solution, so I did this thing instead! This redirecter can be used in all kind of places. You can even add it as a Custom Search Engine in Chrome!

How do I use it?

Add the source book acronym (see below) and page number to this page's URL. For instance, if you want to go to Player's Handbook page 256, add ?PHB256 at the end of the URL.

Valid sources
As of now, only the PHB is covered. WIP for other sources.
Notes on PHB
In the physical book, the section on Weapon describes the weapon properties and is followed with a table showing showing the common weapons. On DDB these are the other way around, so the content flow is slightly different.
The pages following are on Adventuring Gear, and all link to the section Adventuring Gear on DDB, since the individual items are non-linkable.

            
How precise is it?

            
The redirecter is dependent on being able to link to the different sections on DDB. Not all sections are linkable, though.

            
Many pages in the physical books begin with a new section, and thus I can link to the precise place of the page break, provided DDB has a linkable section.

            
Physical book pages that does not begin with a new section, are linked to the closest or most logical corresponding section on DDB.


            
Exact page break

            
Example of exact page breaks are the beginning of new chapters, races, classes etc.


            
Page break above

            
If have deemed the content between the linked section, and the actual physical page break to be noteworthy, you get an alert telling you 
                that the physical page break is above the section you arrive at. I've tried minimizing the use of this, since it's a bit disruptive.


            
Page break below

            
If the physical page break is below the section you arrive at, I don't alert you since you're probably scrolling down anyway.


            
Pages with images

            
Some pages in the physical books consists of only an image. Most of this art is presented somewhere on DDB, but I choose not to redirect to these images.
                One exception are the covers of the books.
